# Учебный проект

## Проект "Вычислитель отличий" 
## Project "Generate Difference"

Утилита для поиска отличий в конфигурационных файлах.

## Возможности утилиты:

* Поддержка форматов: json, yml, ini;
* Генерация отчета в виде plain text, json, stylish(tree);

## SETUP

```sh
$ make install
```

## RUN TESTS

```sh
$ make test
```
